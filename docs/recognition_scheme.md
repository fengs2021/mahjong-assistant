# 雀魂麻将识别方案 v2.0

## 一、整体架构

```
截图(2800×1264) → YOLOv8 TFLite → 56+检测框 → 空间分类 → 手牌/副露/四家河底
                                                ↓
                                         放铳率分析 ← 天凤位统计+河底数据
```

## 二、YOLO 推理链路

```
截图Bitmap(RGB) → preprocess:
  1. ColorMatrix灰度化(setSaturation=0)
  2. 对比度拉伸(min→0, max→255)
  3. 直接resize到640×640 (不用letterbox)
  4. ByteBuffer NHWC [1,640,640,3] float32/255
→ TFLite推理(5.2MB best_float16.tflite)
→ 输出 [1,38,8400]: 4bbox + 34类置信度
→ NMS(IoU=0.3, conf≥0.5) → 最终检测框
```

## 三、空间分类逻辑

### 3.1 底部区域 (y > imgH×0.75) → 手牌+副露

```
底部检测框按x排序 → 找最大间隙
  条件: 最大间隙 > 中位数间隙×1.5 且 > 100px
  ├─ 间隙左边(密集排列): 手牌 (13-14张)
  └─ 间隙右边(稀疏): 副露 (吃碰杠)
```

### 3.2 摸牌定位

```
handEndX = 最后一张高分手牌右边界
drawnX = handEndX + 45px (固定间距)
摸牌区域 = (drawnX, handY, 98, 143)
```

### 3.3 河底四家 (y ≤ imgH×0.75)

| 方位 | 坐标 | 尺寸 |
|------|------|------|
| 自家(南) | (1166,621)-(1621,923) | 455×302 |
| 对家(北) | (1218,138)-(1595,380) | 377×242 |
| 上家(东) | (802,297)-(1190,653) | 388×356 |
| 下家(西) | (1605,302)-(1991,663) | 386×361 |

分类方法：检测框中心点(cx, cy)落在哪个矩形内

## 四、放铳率计算公式

### 4.1 基础铳率 (天凤位统计)

```
基础铳率 = baseDanger[tileId]  (0.015~0.082)
```

| 牌种 | 示例 | 基础铳率 |
|------|------|---------|
| 安全数牌 | 三万(3m) | 0.015 |
| 中张数牌 | 五万(5m) | 0.041 |
| 边张字牌 | 東/南 | 0.07/0.06 |
| dora高危 | 五筒(5p)/発 | 0.082/0.08 |

### 4.2 巡目系数

```
turnFactor = f(剩余手牌数, 估计巡目)
├─ 序盘(>10枚): ×0.5
├─ 中盘(6-10枚): ×1.0
├─ 终盘(3-5枚): ×1.5
└─ 末巡(<3枚): ×2.0
```

### 4.3 剩余枚数修正

```
已见 = 自家副露 + 四家河底计数
未现 = 4 - 已见
危险度 *= 未现/4.0
```

### 4.4 现物检测 (绝对安全)

```
if tile ∈ 任一对手河底:
    危险度 = 0.01  # 仅剩抢杠/双碰极小概率
```

### 4.5 早外检测

```
if tile < 27 (数牌) and 对手前6巡舍弃过同花色:
    危险度 *= 0.2  # 该花色的安全信号
```

### 4.6 筋牌检测

```
if hand包含tile-3和tile+3 (或tile±6):
    危险度 *= 0.15  # 形成筋, 对手几乎不能听
```

### 4.7 无筋生牌 (危险信号)

```
if tile是数牌 and 无筋 and 未在对手河底出现:
    危险度 *= 1.5  # 未现+无筋→高风险
```

### 4.8 壁牌检测

```
if tile±1各已见≥3张:
    危险度 *= 0.5  # 壁牌安全
```

### 4.9 立直修正

```
if 对手立直:
    数牌且危险度>0.3: ×1.5
    字牌且未现: ×2.0  # 立直后生字极危
```

## 五、最终评分

```
totalDanger = min(
    baseDanger × 巡目系数 × 剩余枚数 × 现物(0.01)/早外(0.2)/筋牌(0.15)/无筋(1.5)/壁牌(0.5) × 立直修正,
    2.0
)
```

安全等级：
| 等级 | 分值 | 颜色 |
|------|------|------|
| SAFE | ≤0.3 | 绿 |
| LOW | ≤0.5 | 浅绿 |
| MEDIUM | ≤0.7 | 黄 |
| HIGH | ≤1.0 | 橙 |
| CRITICAL | >1.0 | 红 |

## 六、降级策略

```
YOLO检测失败(模型未加载/崩溃)
  → 模板匹配降级
    ├─ 手牌: 固定坐标(300+236+i×111) + 模板匹配(OpenCV TM_CCOEFF_NORMED)
    ├─ 副露: 扫描右下区域 + 副露模板匹配
    └─ 河底: OpenCV模板匹配自家区域

OpenCV未加载
  → 仅显示"识别不可用"
```
